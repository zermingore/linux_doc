Configuration du prompt

Le prompt (invite de commande?) est configurable.
Par défaut, un prompt ressemble à quelque chose du genre $
C'est heuresement configurable.

Configuration vite fait qui ressemble à quelque chose

Cette configuration:

  1. Gère la couleur, dépendament de si tu es super-user et sur quelle machine
  2. Affiche la valeur de retour (en rouge) de la dernière commande si c'est pas 0
  3. Affiche l'heure
  4. Affiche le chemin jusqu'au dossier courrant
Code à copier-coller dans le fichier de config .bashrc (~/.bashrc) # Gestion de la couleur: # super-user -> rouge # une machine spécifique -> bleu # vert pour le reste (si tu te connecte sur une autre machine, tu sais tout de suite où tu n'est pas) if [ $(hostname) = pi ]; then COLOR_PROMPT=$COLOR_LIGHT_BLUE else if [ $(whoami) = root ]; then COLOR_PROMPT=$COLOR_RED else COLOR_PROMPT=$COLOR_LIGHT_GREEN fi

Comprendre cette configuration

On utilise la couleur rouge dans le cas où on est root (super-user). Pour ce faire on utilise la commande whoami
man whoami
whoami - print effective userid
whoami
jp # par exemple sudo whoami
root
Pour récupérer ce que retourne une commande, on utilise la feature "Command substitution".
man sh ou man bash
Command substitution allows the output of a command to be substituted in place of the command name itself.
