SSH

Secure SHell (SSH) est le tool pour se connecter de manière transparente d'une machine à une autre.
SSH demande vraiment très peu de bande passante (quelque bytes par seconde, on parle de transférer du texte chiffré).
Par conséquent, ce genre de service tourne easy sur, par exemple un télephone, une box internet. En fait une Raspberry c'est déjà vraiment beaucoup... C'est pour ça qu'avec SSH, on peut aussi ouvrir des fenêtres graphiques, si le serveur est le permet et est configuré pour.
Si tu gère un peu le shell, bah bonne nouvelle, il n'y a quasiment rien de plus à savoir: SSH ouvre un shell sur ton serveur que tu peux utiliser sur ton client.

Git

Git est un gestionaire de version.
Comment suivre le fil des changements que tu fais dans du texte sans avoir à tout copier-coller dans tous les sens.
C'est particulièrement utile dans, entre autres, les cas suivants:

  • Voir clairement ce qui là tout de suite dans les 10 dernières minutes à été modifié pour trouver l'explication de pourquoi quelque-chose ne marche soudainement plus / est tombé en marche
  • Comprendre pourquoi et comment exactement quelque-chose à été changé il y a 1 an
  • Pour comparer 2 versions d'un même fichier (ex: un programme fonctionne \o/ ... et maintenant tout est cassé; qu'est-ce qui a exactement changé entre les deux versions?)
  • Éviter de perdre quelque-chose si tu supprime un fichier par hasard, ou si ton disque dur / SSD te lache
  • Pour modifier à plusieurs un même pack de fichier sans avoir à sauter par la fenêtre au moment où il faut tout remettre en commun
    Je sais pas si tu t'es déjà retrouvé à modifier un document Word à plusieurs et à devoir noter où est-ce que t'as rajouté un paragraphe / où est-ce que t'as corrigé une typo ou ce genre de trucs
  • Tenter des trucs YOLO sans prendre le risque de casser quelque-chose
