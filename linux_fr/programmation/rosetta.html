Introduction

Ici, on va écrire le même exemple dans plusieurs languages de programmation différents.
Choisir le language dépend idéalement uniquement de ce qu'on veut faire.
Bien sûr, on a tous des préférences personnelles; ça ne devrait pas être la première raison pour choisir un language.
Utiliser plusieurs languages dans un seul projet est possible.

Exemples

Hello World

Ce programme affiche juste quelque-chose à l'écran; pas plus.

C

      #include  // STandarD Input/Output pour pouvoir utiliser, par exemple printf()

      int main()
      {
        printf("Hello world!\n"); // '\n' saut de ligne explicite

        return 0; // on retourne quelque-chose explicitement parcequ'on est pas des porcs
      }

Shell

      #!/bin/sh

      echo "Hello world!"

Python

      print("Hello world!")

C++

      #include  // Input / Output stream (flux); nécessaire pour utiliser cout

      int main() // on *doit* définir une fonction main()
      {
        std::cout << "Hello world!\n"; // '\n' saut de ligne explicite

        return 0; // on retourne quelque-chose explicitement parcequ'on est pas des porcs
      }
